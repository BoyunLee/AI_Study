* 데이터베이스(DataBase)
- 데이터를 저장하는 공간

* 데이터베이스 관리시스템(Database Management System)
- 데이터를 저장하는 공간을 관리하는 시스템
- 보통 데이터베이스가 설치된 PC(서버 DBserver)

* 관계형 데이터베이스 관리시스템(RDBMS)
- 테이블간의 관계를 통해서 저장공간을 관리하는 시스템

* 주로 사용하는 RDBMS(통칭해서 데이터베이스, DB라고 칭함)
- Oracle : 오라클 회사 / 중대형급 회사 주로 사용
- Mysql : 오라클 회사(2010년도 이후 오라클이 인수) / 중소형급 회사 주로 사용
- MariaDB : MariaDB / 중소형급 회사 주로 사용
- MS-SQL : MS사 / 중대형급 회사 주로 사용(주로 제조회사, 종신계약 되어있는 연구소)

*RDBMS는 모두 소프트웨어(SW)로 인스톨하여 설치하게 됨
- 설치를 하는 순간 DB Server를 구축했다고 표현함
