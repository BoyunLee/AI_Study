<부트스트랩(Bootstrap)>
- 화면에 스타일과 기본적인 자바스크립 코드를 이용해서 코드를 제공해주는 라이브러리(사이트)
- 부트스트랩 사이트에서 다양한 완성된 코드들을 제공하고 있으며, 코드를 복사하여 사용가능
- 코드를 사용하기 위해서는 사용할 HTML 파일 내에, 부트스트랩에서 사용하는 .css 및 .js 파일 환경을 맞춰주어햐 함
- css와 js 파일은 부트스트랩 사이트에서 제공하고 있으며, 추가로 jquery.js 함께 받아야함
* 부트스트랩 접속 : https://getbootstrap.com
* jQuery 접속 URL : https://jquery.com/download
